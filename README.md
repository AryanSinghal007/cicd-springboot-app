1. **Clone the repository:**


   ```bash
   mvn clean install
   ```

3. **Run the application:**
   ```bash
   mvn spring-boot:run
   ```

4. **Access the application:**
   Open your browser and go to [http://localhost:8080/](http://localhost:8080/)
